# careerbot-tanzania

AI WhatsApp Career Assistant - Version 0.1

## Sprint A (Foundation)

This is the production backend foundation.

## Tech Stack
- Node.js 22 LTS
- TypeScript
- Express.js
- PostgreSQL
- Prisma ORM
- Zod

## Setup

1. Copy `.env.example` to `.env` and fill the variables.
2. Run `npm install`
3. Run `npx prisma generate`
4. Run `npm run dev`
